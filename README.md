🖥️ 1. HTML CODE (EASIEST)
✅ Steps:
Open Notepad
Paste HTML code
Save as:
filename.html

👉 Example: profile.html

Double-click file → Opens in browser


🐘 2. PHP CODE (IMPORTANT)
🔧 You NEED server:

Use 👉 XAMPP (most common)

✅ Steps:
Install XAMPP
Open XAMPP Control Panel
Start:
Apache ✅
MySQL ✅
Go to folder:
C:\xampp\htdocs\
Create file:
login.php
Paste PHP code
▶️ Run in browser:
http://localhost/login.php


🗄️ 3. MYSQL (DATABASE)

Use 👉 phpMyAdmin

✅ Steps:
Open:
http://localhost/phpmyadmin
Create database:
test
Run SQL queries


📄 4. XML FILE
✅ Steps:
Open Notepad
Paste XML code
Save as:
student.xml
Double-click → Opens in browser


🎨 5. XML + XSLT
✅ Steps:
Save XML file
Save XSLT file (style.xsl)
Add this in XML:
<?xml-stylesheet type="text/xsl" href="style.xsl"?>
Open XML → Table will display


🌐 6. API PROGRAM (PHP)

Same as PHP steps 👇

👉 Save in:

htdocs/weather.php

👉 Run:

http://localhost/weather.php

⚠️ Replace:

API_KEY

with real key (or just write for exam)


🔐 7. SQL QUERIES

Run inside:

👉 phpMyAdmin

Go to SQL tab
Paste query
Click Run
